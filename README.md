# asdf-canonical-cloud-utils

this is an asdf plugin to install Canonical's [cloud-utils](https://github.com/canonical/cloud-utils).

## Usage

1. Install `cloud-init`:

    ```shell
    asdf plugin add cloud-utils https://github.com/alexanderankin/asdf-canonical-cloud-utils.git
    asdf install cloud-utils latest
    (cd; asdf set cloud-utils latest;)
    ```
